ශේෂ ප්‍රමේය හා සාධක ප්‍රමේය

A/L ගණිතය - ටියුට් එක

1. ශේෂ ප්‍රමේය (Remainder Theorem)

f(x) බහුපදයක් (x-a) මගින් බෙදූ විට ලැබෙන ශේෂය f(a) වේ.

සාධනය:

f(x) බහුපදය (x-a) මගින් බෙදූ විට:

  • f(x) = (x-a)q(x) + R
  • මෙහි q(x) යනු ලබ්ධිය සහ R යනු ශේෂයයි
  • x = a ආදේශ කරමු: f(a) = (a-a)q(a) + R
  • f(a) = 0 + R
  • එබැවින් R = f(a)
උදාහරණ 1:

f(x) = x³ + 2x² - 4x + 2 බහුපදය (x+2) මගින් බෙදූ විට ශේෂය සොයන්න.

  • (x+2) = (x-(-2))
  • ශේෂය f(-2)
  • f(-2) = (-2)³ + 2(-2)² - 4(-2) + 2
  • f(-2) = -8 + 8 + 8 + 2 = 10
  • ශේෂය 10 වේ
උදාහරණ 2:

f(x) = 2x³ - x² + 4 බහුපදය (x-1) මගින් බෙදූ විට ලබ්ධිය සහ ශේෂය සොයන්න.

  • ශේෂය f(1) = 2(1)³ - (1)² + 4 = 2 - 1 + 4 = 5
  • බෙදුම් ඇල්ගොරිතමය භාවිතයෙන් ලබ්ධිය සොයමු:
    •       2x² + x + 1
x-1 ) 2x³ - x² + 0x + 4
      2x³ - 2x²
      --------
          x² + 0x + 4
          x² - x
          --------
              x + 4
              x - 1
              --------
                  5
  • ලබ්ධිය 2x² + x + 1
  • ශේෂය 5

සැලකිය යුතුයි: f(x) බහුපදය (ax+b) මගින් බෙදූ විට ශේෂය සොයන්න නම්, x = -b/a ආදේශ කරන්න.

2. සාධක ප්‍රමේය (Factor Theorem)

f(a) = 0 නම්, (x-a) යනු f(x) හි සාධකයකි.

සාධනය:

  • ශේෂ ප්‍රමේයය අනුව, f(x) බහුපදය (x-a) මගින් බෙදූ විට ශේෂය f(a) වේ
  • f(a) = 0 නම්, ශේෂය 0 වේ
  • ශේෂය 0 වන බැවින්, (x-a) යනු f(x) හි සාධකයකි
  • එබැවින් f(x) = (x-a)q(x)
උදාහරණ 1:

f(x) = x³ - x - 6 බහුපදයේ (x-2) සාධකයක් දැයි පරීක්ෂා කරන්න.

  • f(2) ගණනය කරමු
  • f(2) = 2³ - 2 - 6 = 8 - 2 - 6 = 0
  • f(2) = 0 බැවින්, (x-2) යනු f(x) හි සාධකයකි
උදාහරණ 2:

f(x) = x³ + 2x² + px + q බහුපදයේ (x-1) හා (x+2) සාධක නම්, p හා q සොයන්න.

  • f(1) = 0 නම්: 1 + 2 + p + q = 0 → p + q = -3
  • f(-2) = 0 නම්: -8 + 8 - 2p + q = 0 → -2p + q = 0 → q = 2p
  • ඉහත සමීකරණ විසඳීමෙන්: p = -1, q = -2

3. ඉතිරි සාධක සෙවීම

පියවර:

  1. f(x) බහුපදයේ (x-a) සාධකයක් නම්, f(x) = (x-a)g(x) ලෙස ලියත හැක
  2. g(x) = Ax^(n-1) + Bx^(n-2) + ... සාධාරණ ආකාරයේ බහුපදයක් බව උපකල්පනය කරන්න
  3. දෙපැත්ත ගුණ කර සංගුණක සංසන්දනය කරන්න
  4. A, B, C... අගයන් සොයාගන්න
උදාහරණ:

f(x) = x³ - 4x² + x + 6 බහුපදයේ (x-2) සාධකයක් නම්, ඉතිරි සාධකය සොයන්න.

  1. f(x) = (x-2)(ax² + bx + c) ලෙස ලියමු
  2. දෙපැත්ත ගුණ කරමු: x³ - 4x² + x + 6 = (x-2)(ax² + bx + c)
  3. සංගුණක සංසන්දනය:
    • x³ සංගුණකය: a = 1
    • x² සංගුණකය: -2a + b = -4, එබැවින් b = -2
    • x¹ සංගුණකය: -2b + c = 1, එබැවින් c = -3
  4. ඉතිරි සාධකය: x² - 2x - 3 = (x-3)(x+1)
  5. එබැවින් f(x) = (x-2)(x-3)(x+1)

4. (x-a)² මගින් බෙදීම

ක්‍රමය:

  1. පළමුව f(x) බහුපදය (x-a) මගින් බෙදා ලබ්ධිය q(x) සහ ශේෂය r₁ සොයන්න
  2. ඊළඟට q(x) බහුපදය (x-a) මගින් බෙදා ලබ්ධිය p(x) සහ ශේෂය r₂ සොයන්න
  3. එවිට f(x) = (x-a)²p(x) + (x-a)r₂ + r₁
උදාහරණ:

f(x) = 2x³ - x² + 4 බහුපදය (x-1)² මගින් බෙදූ විට ශේෂය සොයන්න.

  1. පළමුව (x-1) මගින් බෙදීමෙන්:
    • ලබ්ධිය: 2x² + x + 1
    • ශේෂය: 5
  2. ඊළඟට ලබ්ධිය (x-1) මගින් බෙදීමෙන්:
    • ලබ්ධිය: 2x + 3
    • ශේෂය: 4
  3. එබැවින්:
    • f(x) = (x-1)²(2x + 3) + (x-1)(4) + 5
    • ශේෂය: 4(x-1) + 5 = 4x - 4 + 5 = 4x + 1

5. (x-a)(x-b) මගින් බෙදීම

ක්‍රමය:

  1. (x-a)(x-b) = x² - (a+b)x + ab ලෙස ගුණ කරන්න
  2. බහුපදය x² - (a+b)x + ab මගින් බෙදා ශේෂය Ax + B ආකාරයේ බව පෙන්වන්න
  3. A, B අගයන් සොයන්න
උදාහරණ:

f(x) = x³ + 2x² + 4x + 2 බහුපදය (x²-1) මගින් බෙදූ විට ශේෂය සොයන්න.

  1. (x²-1) = (x-1)(x+1)
  2. පළමුව (x-1) මගින් බෙදීමෙන්:
    • ලබ්ධිය: x² + 3x + 7
    • ශේෂය: 9
  3. ඊළඟට ලබ්ධිය (x+1) මගින් බෙදීමෙන්:
    • ලබ්ධිය: x + 2
    • ශේෂය: 5
  4. එබැවින්:
    • ශේෂය: 5(x-1) + 9 = 5x - 5 + 9 = 5x + 4

6. බහුපද සමීකරණ විසඳීම

ක්‍රමය:

  1. f(x) = 0 සමීකරණයේ සාධක සොයන්න
  2. සෑම සාධකයක්ම 0 ට සමාන කර මූල සොයන්න
උදාහරණ:

x³ - 6x² + 11x - 6 = 0 සමීකරණය විසඳන්න.

  1. f(1) = 1 - 6 + 11 - 6 = 0 බැවින්, (x-1) සාධකයකි
  2. f(x) = (x-1)(x² - 5x + 6)
  3. x² - 5x + 6 = (x-2)(x-3)
  4. එබැවින් f(x) = (x-1)(x-2)(x-3)
  5. මූල: x = 1, x = 2, x = 3

පුහුණු ගැටලු

ගැටලුව 1:

f(x) = x⁴ - 5x³ + 3x² + 5x - 2 බහුපදය (x-2) මගින් බෙදූ විට ශේෂය සොයන්න.

ශේෂ ප්‍රමේයය අනුව, ශේෂය f(2) වේ.

f(2) = 2⁴ - 5(2)³ + 3(2)² + 5(2) - 2

f(2) = 16 - 40 + 12 + 10 - 2

f(2) = -4

එබැවින් ශේෂය -4 වේ.

ගැටලුව 2:

f(x) = 2x³ - 3x² - 11x + 6 බහුපදයේ සාධක සොයන්න.

පළමුව ඇතැම් අගයන් සඳහා f(x) = 0 දැයි පරීක්ෂා කරමු:

f(1) = 2 - 3 - 11 + 6 = -6 (සාධකයක් නොවේ)

f(-1) = -2 - 3 + 11 + 6 = 12 (සාධකයක් නොවේ)

f(2) = 16 - 12 - 22 + 6 = -12 (සාධකයක් නොවේ)

f(-2) = -16 - 12 + 22 + 6 = 0

f(3) = 54 - 27 - 33 + 6 = 0

f(-2) = 0 හා f(3) = 0 බැවින්, (x+2) හා (x-3) සාධක වේ.

f(x) = 2x³ - 3x² - 11x + 6 = 2(x+2)(x-3)(x-?)

සංගුණක සංසන්දනය කරමින් අවසාන සාධකය (x-1/2) බව සොයාගත හැක.

එබැවින් f(x) = 2(x+2)(x-3)(x-1/2)

ගැටලුව 3:

f(x) = x⁴ + x³ - 7x² - x + 6 බහුපදය (x-1)(x+2) මගින් බෙදූ විට ශේෂය සොයන්න.

ශේෂය Ax + B ආකාරයේ වේ.

x = 1 විට: f(1) = 1 + 1 - 7 - 1 + 6 = 0

x = -2 විට: f(-2) = 16 - 8 - 28 + 2 + 6 = -12

A(1) + B = 0 → A + B = 0

A(-2) + B = -12 → -2A + B = -12

මේ සමීකරණ විසඳීමෙන්: A = 4, B = -4

එබැවින් ශේෂය 4x - 4 වේ.
