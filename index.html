f(x) = 3x³ - 4x² + 2x - 1 බහුපදය (x+1) මගින් බෙදූ විට ශේෂය සොයන්න.

ශේෂ ප්‍රමේයය අනුව, ශේෂය f(-1) වේ.

f(-1) = 3(-1)³ - 4(-1)² + 2(-1) - 1

= -3 - 4 - 2 - 1 = -10

එබැවින් ශේෂය -10 වේ.

ගැටලුව 5:

f(x) = x³ - 7x + 6 බහුපදයේ සාධක සොයන්න.

f(1) = 1³ - 7(1) + 6 = 1 - 7 + 6 = 0

f(1) = 0 බැවින්, (x-1) සාධකයකි.

f(x) = (x-1)(x² + x - 6)

x² + x - 6 = (x+3)(x-2)

එබැවින් f(x) = (x-1)(x+3)(x-2)

ගැටලුව 6:

x³ + 2x² - 5x - 6 = 0 සමීකරණය විසඳන්න.

f(1) = 1³ + 2(1)² - 5(1) - 6 = 1 + 2 - 5 - 6 = -8 (සාධකයක් නොවේ)

f(-1) = (-1)³ + 2(-1)² - 5(-1) - 6 = -1 + 2 + 5 - 6 = 0

f(-1) = 0 බැවින්, (x+1) සාධකයකි.

f(x) = (x+1)(x² + x - 6)

x² + x - 6 = (x+3)(x-2)

f(x) = (x+1)(x+3)(x-2)

මූල: x = -1, x = -3, x = 2
